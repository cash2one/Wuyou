原省考试院专家解析：
粤50%考生将受高考改革冲击

最近，高考改革成为广大高中生家长关注的头等大事。值得注意的是，对于高三学生来说，他们首先受这一大刀阔斧改革“牵连”。其中之一，那就是自主招生，以下三大巨变有可能导致明年高考竞争更为激烈，至少超一半考生受改革冲击。其中，自招更成为广大考生的必争之地！

变化一：明年考试时间推迟到高考后，或在六月中旬，高考录取前进行。

影响：专家分析，改革后，自招考试对中学的教学冲击会减少，对学生的备考有好处。但是，这样一来考生将会有更多的精力投入到高考备考中，因此明年高考竞争或会更激烈。

变化二：自招高校不得采用联考方式或组织专门培训。

影响：2015年开始，北约、华约和卓越联盟等三大自主招生联盟将消失。考生可能要根据高考成绩成绩，选择一两所目标院校参加自主选拔。

欲知高考改革对广东高三学生影响详细信息，原广东省教育考试院专家、盈速教育研究院院长邱健仕及更多来自知名高校、高中的权威人士，将在“校园+”APP为大家独家解读与支招。
